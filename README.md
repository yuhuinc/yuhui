# YuhUI

## Goals

Accessible React components

- Force translations for copy
- Follow web accessibility standards as much as possible

## Getting started

### Using the package

```bash
$ npm install @yuhuinc/yuhui
$ yarn add @yuhui/yuhui
```

### Developing

```bash
$ yarn storybook
```
