# Buttons

Accessible buttons.

**Available components**:

- [ButtonPrimary](#buttonprimary)

**How it is built**

- Uses `reakit/Button` ([link](https://reakit.io/docs/button/)) under the hood.
- Wrapped in `withContent`.

## ButtonPrimary

### Example

```javascript
<ButtonPrimary contentKey="demo.Button" onClick={() => setLang("en")} />
```
